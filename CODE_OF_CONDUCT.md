# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Standard

Be respectful, constructive, and patient. Assume good intent, but take responsibility for impact.

Examples of welcome behavior:

- giving clear, actionable feedback
- being kind to new contributors
- explaining tradeoffs without condescension
- respecting different experience levels and backgrounds

Unacceptable behavior includes harassment, insults, discrimination, threats, or publishing private information.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block participants who violate this code of conduct.

Report concerns privately to the project maintainer.
